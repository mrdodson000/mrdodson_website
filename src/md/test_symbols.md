# Greek Letters and Symbols

## Common Greek Letters

Lowercase: $\alpha, \beta, \gamma, \delta, \epsilon, \theta, \lambda, \mu, \pi, \sigma, \tau, \phi, \omega$

Uppercase: $\Gamma, \Delta, \Theta, \Lambda, \Sigma, \Phi, \Omega$

## Set Theory

The empty set is denoted $\emptyset$ or $\{\}$.

Set operations:
- Union: $A \cup B$
- Intersection: $A \cap B$
- Subset: $A \subseteq B$
- Element: $x \in A$

## Logic Symbols

$$
\forall x \in \mathbb{R}, \exists y \in \mathbb{R}: x < y
$$

This means: for all $x$ in the real numbers, there exists a $y$ such that $x < y$.

## Special Sets

- Natural numbers: $\mathbb{N}$
- Integers: $\mathbb{Z}$
- Rationals: $\mathbb{Q}$
- Reals: $\mathbb{R}$
- Complex: $\mathbb{C}$

The cardinality of a set $A$ is denoted $|A|$ or $\#A$.
